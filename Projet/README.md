# Naruto vs Sasuke - README

## Démarrage rapide

1. **Cloner ou télécharger le projet**

2. **Se placer dans le dossier du projet**
```bash
cd /home/habib/Bureau/venv
```

3. **Créer un environnement virtuel (si besoin)**
```bash
python3 -m venv .venv
```

4. **Activer l'environnement virtuel**
```bash
source .venv/bin/activate
```

5. **Se placer dans le projet**
```bash
cd Projet/
```

6. **Installer les dépendances nécessaires**
```bash
pip install pygame opencv-python
```

7. **Lancer le jeu**
```bash
python naruto_vs_sasuke.py
```

- Si d'autres modules sont nécessaires, installez-les avec `pip install nom_du_module`.
